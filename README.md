# Tarif Radar

Современный адаптивный сайт для подбора и сравнения тарифов операторов и MVNO по регионам России.

## Что уже реализовано

- Автоопределение геолокации пользователя и подбор тарифов по региону.
- Блок **«Подберем лучший тариф»** с простыми условиями:
  - бюджет,
  - нужные ГБ,
  - нужные минуты,
  - нужные SMS,
  - доп. опции (бесплатные соцсети, роуминг).
- Автоматический выбор **лучшего тарифа в регионе** по скорингу выгодности.
- Режим «Показать все тарифы в регионе» для просмотра полного списка без ограничений по условиям.
- Карточки тарифов с логотипами операторов и переходом на страницу подключения у оператора.
- Светлая/тёмная тема через переключатель-тумблер (сохранение выбора).
- Режим модератора: inline-редактирование, удаление, импорт/экспорт JSON, localStorage.
- SEO-блок с релевантным текстом для запросов про лучший тариф в Москве и регионах.
- Добавлен каркас парсера тарифов (`parser/run_parser.py`) и инструкция по хостингу (`parser/DEPLOY.md`).

## Запуск сайта

```bash
python3 -m http.server 4173
```

Откройте: http://localhost:4173

## Запуск парсера

```bash
python3 parser/run_parser.py --region Москва --out data/tariffs.json
```

Подробно про хостинг и cron: `parser/DEPLOY.md`.

## Структура

- `index.html` — разметка UI и SEO-блок.
- `styles.css` — адаптивный дизайн, темы и визуальные компоненты.
- `app.js` — логика подбора, геолокация, рендер, модерация.
- `data/tariffs.json` — стартовые данные тарифов.
- `assets/logos/` — локальные SVG-лого операторов.
- `parser/run_parser.py` — скрипт парсера/агрегации тарифов.
- `parser/DEPLOY.md` — инструкция по хостингу парсера.
